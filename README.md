# Geese Gone Galactic (v3)

Fresh restart. The real deliverable is a strict, self-improving **harness** that
bootstraps a locally-based game-dev AI (**Icarus**), used to build a great
Geese Gone Galactic. Engine: **Godot** (3D low-poly, 2.5D fixed iso camera).

See `docs/PLAN.md` for the full architecture. This repo is intentionally clean;
the archived Unity v1/v2 lives separately and is not built upon.
